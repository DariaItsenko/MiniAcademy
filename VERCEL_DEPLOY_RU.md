# 🚀 Как развернуть приложение на Vercel

## Пошаговая инструкция на русском

---

## Шаг 1: Подготовка проекта

### Убедитесь что у вас есть:
- ✅ Иконки: `/public/icon-192.png` и `/public/icon-512.png`
- ✅ Все файлы проекта
- ✅ Файл `package.json`

---

## Шаг 2: Создать аккаунт на Vercel

1. **Перейдите на:** https://vercel.com

2. **Нажмите "Sign Up"** (Зарегистрироваться)

3. **Выберите способ регистрации:**
   - GitHub (рекомендуется, если код на GitHub)
   - GitLab
   - Bitbucket
   - Email

4. **Подтвердите email** (если используете email)

---

## Шаг 3: Загрузить проект

### Вариант A: Через GitHub (рекомендуется)

1. **Загрузите код на GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/ваш-username/ваш-repo.git
   git push -u origin main
   ```

2. **В Vercel:**
   - Нажмите **"Add New..."** → **"Project"**
   - Выберите **"Import Git Repository"**
   - Выберите ваш репозиторий
   - Нажмите **"Import"**

### Вариант B: Загрузить файлы напрямую

1. **Установите Vercel CLI:**
   ```bash
   npm install -g vercel
   ```

2. **В папке проекта выполните:**
   ```bash
   vercel
   ```

3. **Следуйте инструкциям:**
   - Login (войти)
   - Set up and deploy (настроить и развернуть)
   - Подтвердите настройки (нажимайте Enter)

---

## Шаг 4: Настройки развертывания

Vercel автоматически определит настройки:

```
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
```

✅ **Не меняйте ничего** - просто нажмите **"Deploy"**!

---

## Шаг 5: Дождитесь развертывания

- Займет **1-2 минуты**
- Вы увидите прогресс в реальном времени
- После завершения увидите: **"Congratulations!"** 🎉

---

## Шаг 6: Получите ссылку на приложение

После развертывания вы получите URL:

```
https://ваш-проект.vercel.app
```

Например:
```
https://elementary-learning.vercel.app
```

---

## Шаг 7: Настроить кастомный домен (опционально)

### Если хотите свой домен (например: learning.myschool.com):

1. **В Vercel:**
   - Откройте ваш проект
   - Нажмите **"Settings"** → **"Domains"**
   - Введите ваш домен
   - Нажмите **"Add"**

2. **В вашем регистраторе доменов:**
   - Добавьте CNAME запись
   - Укажите на `cname.vercel-dns.com`
   - Подождите 5-10 минут

3. **Готово!** Ваше приложение доступно на вашем домене!

## 🔄 Как обновлять приложение

### Если код на GitHub:
1. Внесите изменения в код
2. Сделайте commit и push:
   ```bash
   git add .
   git commit -m "Обновление"
   git push
   ```
3. Vercel **автоматически** развернет новую версию!
4. Студенты получат обновление автоматически (в течение 24 часов)

### Если используете Vercel CLI:
```bash
vercel --prod
```

## ✅ Проверка развертывания

### Проверьте что все работает:

1. **Откройте ваш URL** в браузере
2. **Проверьте иконки:**
   - Откройте DevTools (F12)
   - Вкладка **Application** → **Manifest**
   - Должны быть видны ваши фиолетовые иконки с книгой
3. **Попробуйте установить:**
   - Нажмите кнопку "📱 Install App" внизу справа
   - ИЛИ используйте меню браузера
4. **Войдите в приложение:**
   - Создайте тестовый аккаунт
   - Проверьте все функции

## 🆘 Решение проблем

### "Build failed" (Ошибка сборки)

**Проверьте:**
```bash
# Попробуйте собрать локально
npm install
npm run build

# Если есть ошибки - исправьте их
# Потом загрузите обновленный код
```

### "Icons not showing" (Иконки не отображаются)

**Убедитесь что:**
- Файлы `/public/icon-192.png` и `/public/icon-512.png` существуют
- Названия файлов точно совпадают (lowercase, с дефисом)
- Файлы загружены в репозиторий

### "App not installing" (Приложение не устанавливается)

**Проверьте:**
- Используете HTTPS (Vercel автоматически дает HTTPS ✓)
- Посетили страницу минимум 2 раза
- Используете поддерживаемый браузер (Chrome, Safari, Edge)
